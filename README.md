# Sync Lyrics

**Sync Lyrics** is a lightweight web app that helps you synchronize song lyrics for YouTube captions with ease.
It is built using **Google AI Studio (Gemini)** and hosted on **Cloudflare Pages**.

## Features

* Sync lyrics with precise timestamps
* Import and export SRT files with synced timing
* Fully responsive design for desktop and mobile

## Run Locally

### Prerequisites

* [Bun](https://bun.sh/)

### Steps

1. Install dependencies:

   ```bash
   bun install
   ```
2. Start the development server:

   ```bash
   bun run dev
   ```
